# Veilkeeper

A turn-based tactical RPG where positioning, abilities, and squad composition shape every battle.

## About

Veilkeeper is a squad-based tactical RPG focused on meaningful combat decisions, positioning, and emergent gameplay. Each encounter challenges players to adapt their strategy, manage their team, and make calculated tactical choices.

The game emphasizes clarity, readability, and strategic depth without overwhelming complexity.

Currently in active development.

## Core Pillars

- Turn-based tactical combat  
- Squad-based decision making  
- Positioning-driven gameplay  
- Emergent encounter design  
- Readable, intentional combat systems  
- Strategic choices over stat stacking
